Standard Printer Options
========================

How to obtain printer options?

1. file = SCR(`Read(.cups.[printers|classes].<name>.ppd)) gives path to
   printer/class ppd. 
2. SCR(`Read(.ppd.settings,file)) gives list of options.
3. SCR(`Read(.cups.[printers|classes].<name>.options)) gives list of printer
   options that were already changed.
